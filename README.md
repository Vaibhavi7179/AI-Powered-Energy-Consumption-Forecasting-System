# ⚡ AI-Powered Energy Consumption Forecasting System

## 📌 Project Overview
This project uses Machine Learning to predict future energy consumption based on historical data. It helps in understanding energy usage patterns and supports efficient energy management.

---

## 🎯 Objective
- Forecast future electricity consumption
- Analyze usage trends
- Build an interactive prediction system

---

## 📊 Dataset
- Time-based energy consumption data
- Features:
  - Datetime
  - Energy (target variable)

---

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Google Colab

---

## 🤖 Machine Learning Model
- Model: MLP Regressor
- Type: Supervised Learning (Regression)

---

## 🔄 Workflow
1. Data Loading
2. Data Preprocessing
3. Feature Engineering (Hour, Day)
4. Model Training
5. Evaluation
6. Prediction
7. Visualization
8. UI (Interactive prediction using widgets)

---

## 🖥️ Features
- Predict energy consumption based on time inputs
- Visualization of actual vs predicted values
- Interactive UI for real-time prediction

---

## 📈 Output
- Energy prediction values
- Graphs showing trends and model performance

---

## 🚀 How to Run
1. Open notebook in Google Colab
2. Upload dataset file (`energy_dataset.csv`)
3. Run all cells step-by-step
4. Use UI sliders to predict energy
